# 流、元素与基本尺寸

两类：块级元素和内联元素

## 块级元素
常见的有：<div> 、<li>、<table>

li: display: list-item
table: display : table

均为块级元素，因为他们都符合块级元素的基本特征：即是一个水平流上只能单独显示一个元素，多个块级元素则换行显示
注意：块级元素和display为block的元素不是一个概念

正是由于“块级元素”具有换行特性，因此理论上它都可以配合clear 属性来清除浮动带来的影响
